# 📚 AI-Powered Study Planner

An intelligent study planning application built in Python that automates task scheduling, prioritization, and progress tracking for students.

Designed to improve productivity through structured task management and smart scheduling logic.

---

## 🚀 Features

- 📅 Automated task scheduling
- 🧠 Priority-based task organization
- 📊 Progress tracking system
- ⏰ Reminder functionality
- 🖥️ Interactive GUI for visualization
- 🔌 Designed for optional Arduino hardware integration

---

## 🛠️ Tech Stack

- **Python**
- Tkinter (GUI)
- Custom scheduling algorithms
- Modular backend architecture

---

## 🧠 Project Motivation

Students often struggle with time management and workload distribution.  
This project explores how automation and structured logic can help optimize study efficiency.

---

## 🏗️ Architecture Overview

- Task Manager Module
- Scheduling Engine
- GUI Layer
- (Optional) Hardware Integration Layer

The backend logic was built first, with the GUI layered on top for visualization and interaction.
